## 📖 Overview

PatientCare is a web-based healthcare appointment booking system designed to simplify interactions between patients and doctors. The platform enables patients to search for doctors, book appointments, manage schedules, and access healthcare services through a modern and user-friendly interface.

The project aims to reduce manual appointment scheduling and improve accessibility to healthcare services through digital transformation.

## ⚙️ Installation Guide

### 1. Clone Repository

```bash
git clone https://github.com/your-username/patientcare.git
```

### 2. Navigate to Project Folder

```bash
cd patientcare
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Configure Environment Variables

Create a `.env` file:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### 5. Run Development Server

```bash
npm run dev
```

Application runs on:

```text
http://localhost:5173
```
